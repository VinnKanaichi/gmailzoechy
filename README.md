## 🌐 Zoechy Gamestore Website — Collaboration with Furina Team

This repository contains the source code for the **Zoechy Gamestore** website, a game top-up and digital products store platform.

### 📌 Project Background

**Zoechy Gamestore** reached out to me with a vision to establish a professional and modern web presence. They needed a website that could effectively showcase their services, deliver a seamless user experience, and build trust within the gaming community. 

This project was developed entirely by me, **Vinn**, in collaboration with **Furina Team**. Together, we brought their vision to life — from initial concept to final deployment. This partnership highlights my ability to work in a team environment while delivering high-quality web solutions as a self-taught developer.

### ✨ Features

- **Modern UI/UX:** Clean, responsive design optimized for both desktop and mobile devices.
- **Smooth Navigation:** Intuitive layout that makes browsing effortless.
- **Interactive Elements:** Dynamic content powered by JavaScript.
- **Backend API:** A simple Node.js API for handling requests.
- **Telegram Integration:** Connected with Telegram Bot API for real-time order notifications.
